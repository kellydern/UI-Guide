# UI-Guide

Sojern UI Guide

This is a guide to Sojern’s main UI elements. In order to keep our current and future product development on-brand, we’ve created this UI Style Guide as a central library of where we keep all of our up-to-date UI components, which include the following: brand overview, brand guidelines, brand assets, grids, code snippets, developer guidelines and much, much more.

Project Setup and Installation

1. Clone this repository
2. Install
3. Download <a href=“https://github.com/mathieudutour/git-sketch-plugin”>The Git/Sketch plugin</a>
4. To generate files for preview, select the plugin then “Generate files for pretty diffs”
5. Save both Sketch file and artboard export.
